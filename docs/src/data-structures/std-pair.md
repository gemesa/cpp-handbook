# `std::pair`

## Explanation

Holds exactly two values of potentially different types.

See [`std::pair`](https://en.cppreference.com/w/cpp/container/pair.html).

Memory: inline (stack if local). Size = sizeof(T1) + sizeof(T2) + padding.
